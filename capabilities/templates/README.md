# Capability templates

Each YAML file in this directory is a curated `CapabilitySpec` users can pick in **Recommended mode**. Anyone can fork or author new templates.

V1 ships 5 templates (filled in during Phase 3):

- `groundedness.yaml`
- `instruction-following.yaml`
- `code-correctness.yaml`
- `uncertainty-calibration.yaml`
- `multi-step-reasoning.yaml`

The structure matches the `CapabilitySpec` Pydantic schema in `packages/capability-compiler`.
